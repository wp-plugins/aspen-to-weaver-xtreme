Aspen to Weaver Xtreme Settings Converter

== Description ==

This plugin will non-destructively convert your Aspen Theme settings to Weaver Xtreme Theme settings.

Please read instructions on the Help tab.


== Installation ==

Please use the Plugin Add Plugin page to install this plugin.

Read the instructions on the Help tab.
